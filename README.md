# Play2Learn

Play2Learn is a browser-based educational web platform built with HTML, CSS, and JavaScript. The project is designed to provide interactive learning activities for students, including a Math Facts game that helps users practice arithmetic skills in an engaging way.

## Features

* Multi-page educational website layout
* Interactive Math Facts game
* Timer-based gameplay
* Score tracking
* Multiple operation modes
* On-screen keypad input
* Dynamic UI updates with JavaScript
* Responsive page structure
* Organized project files using HTML, CSS, and JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## What I Learned

While building Play2Learn, I practiced structuring a larger front-end project, organizing JavaScript logic, updating the DOM based on user interaction, and managing game state. I also gained more experience with responsive layouts, form/input handling, and writing cleaner, more maintainable code.
